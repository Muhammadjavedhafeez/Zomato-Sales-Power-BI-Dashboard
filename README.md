# 🍽️ Zomato Sales Analysis Dashboard \| Power BI Project

## 📌 Project Summary

The Zomato Sales Power BI Dashboard analyzes sales performance, outlet
distribution, product trends, and customer ratings. This project
transforms raw sales data into meaningful business insights using
interactive visualizations and DAX-driven KPIs.

The dashboard enables stakeholders to monitor performance metrics,
identify high-performing outlet categories, and evaluate sales trends
across location tiers and time.

------------------------------------------------------------------------

## 🎯 Business Objectives

-   Analyze overall sales and item distribution performance\
-   Compare outlet performance by size, type, and location tier\
-   Identify top-performing product categories\
-   Evaluate customer satisfaction using rating metrics\
-   Track sales trends based on outlet establishment year

------------------------------------------------------------------------

## 📊 Key Performance Indicators (KPIs)

-   **Total Sales:** \$1.2M\
-   **Total Items Sold:** 8,523\
-   **Average Sales per Item:** \$141\
-   **Average Rating:** 3.92

------------------------------------------------------------------------

## 📈 Dashboard Insights

### 🔹 Outlet Analysis

-   Tier 3 outlets contribute the highest revenue share\
-   Supermarket Type 1 dominates total item sales\
-   Medium-sized outlets generate the largest sales portion

### 🔹 Product Performance

-   Regular fat products contribute more to total sales than Low Fat\
-   Fruits, Snacks, and Household items are among the top-performing
    categories

### 🔹 Trend Analysis

-   Sales peaked significantly for outlets established around 2018\
-   Stable performance observed in recent establishment years
